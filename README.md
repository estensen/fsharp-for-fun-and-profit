# fsharp-for-fun-and-profit

## Install
Install [mono](https://www.mono-project.com/download/stable/)

## Compile and run
```
$ $ fsharpc Hello.fsx
$ mono Hello.exe
Hello, F#
```

## Interactive shell
```
$ fsharpi
> printfn "yo";;
yo
val it : unit = ()
```

## Resources
Many examples are inspired and adapted from [F# for fun and profit](https://fsharpforfunandprofit.com/)
